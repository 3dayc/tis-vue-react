ESNext 관련 학습할 내용

ESNext는 ES2015, ES2016, ES2017, ES2019, ... 를 지칭하는 용어다.

학습 주제

  • var vs let vs const
  • 템플릿 리터럴
  • 디폴트 매개변수
  • 가변 매개변수
  • function 함수 vs 화살표 함수
  • 분해 할당(Destructure assignment)
  • ES6 신규연산자: rest 연산자 vs spread 연산자
    • rest 연산자
    • spread 연산자
  • 객체 생성
    • 생성자로 객체 생성
    • class로 객체 생성
  • 비동기
    • 콜백
    • 프로미스(Promise)
    • async, await
  • For … Of 루프
  • 개선된 객체 표현식
  • 모듈 import 와 export
