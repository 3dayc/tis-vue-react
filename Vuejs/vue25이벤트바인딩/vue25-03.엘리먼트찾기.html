태그 찾기 API

      +───────────────────────────────────+─────────────────────+
      | 메소드 이름                                                          | 설명                                     |
      +───────────────────────────────────+─────────────────────+
      | document.getElementById(아이디)                                      | 아이디를 사용해 객체 선택. 한개 리턴     |
      | document.getElementsByName(이름)                                     | name 속성으로 객체 선택. 여러개 리턴     |
      | document.getElementsByClassName(클래스)                              | class 속성으로 객체 선택. 여러개 리턴    |
      | document.getElementsByTagName(이름)                                  | element 이름으로 객체 선택. 여러개 리턴  |
      | document.querySelectorAll(선택자)                                    | 선택자로 여러 개의 객체 선택             |
      | document.querySelector(선택자)                                       | 선택자를 사용해 객체 선택                |
      +───────────────────────────────────+─────────────────────+
      | document.querySelector("div.user-panel.main input[name='login']");                                              |
      +───────────────────────────────────+─────────────────────+
