인라인 이벤트 핸들러 설정

            태그 내의 속성(attribute)으로 이벤트 핸들러 선언하는 방법
            웹 페이지의 가장 기본적인 연결 방식
            태그 속성에 자바스크립트 코드를 넣어 이벤트를 연결

고전 이벤트 핸들러 설정

            JavaScript의 코드 내에서 이벤트 핸들러 선언하는 방식
            이벤트명은 모두 소문자로 기술 - onLoad, onClick은 불가
            이벤트 하나에 이벤트 핸들러 하나만 연결
이벤트 핸들러 작성

표준 이벤트 핸들러 설정

            한 번에 여러 가지 이벤트 핸들러 추가 가능
            추가: addEventListener(eventName, handler);
            삭제: removeEventListener(eventName, handler);
