  1. 인라인 방식 핸들러
  2. 고전 방식 핸들러 
                document.onClick = function(e){
                document.body.style.backgroundColor = "#FFC926";
            };
  3. 표준 방식 핸들러 
                var changeColor = function(e){
                document.body.style.backgroundColor = "#FFC926";
            };
            document.addEventListener('click', changeColor, true);

JavaScript의 이벤트 객체

  1. AnimationEvent For CSS animations
  2. ClipboardEvent For modification of the clipboard
  3. DragEvent For drag and drop interaction
  4. FocusEvent For focus-related events
  5. HashChangeEvent For changes in the anchor part of the URL
  6. InputEvent For user input
  7. KeyboardEvent For keyboard interaction
  8. MouseEvent For mouse interaction
  9. PageTransitionEvent For navigating to, and away from, web pages
  10. PopStateEvent For changes in the history entry
  11. ProgressEvent For the progress of loading external resources
  12. StorageEvent For changes in the window's storage area.
  13. TouchEvent For touch interaction
  14. TransitionEvent For CSS transitions
  15. UiEvent For user interface interaction
  16. WheelEvent For mousewheel interaction
