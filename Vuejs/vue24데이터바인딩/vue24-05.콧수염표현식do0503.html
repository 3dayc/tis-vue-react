콧수염 표현식, {{ }} 에서는  첫째, 콧수염 표현식 에서는 자바스크립트를 사용할 수 있다. 둘째, 콧수염 표현식 에서는 변수 선언문과 분기문은 사용 할 수 없다. 세째, 콧수염 표현식 에서는 삼항 연산자는 사용 가능하다. 네째, 복잡한 연산은 computed 속성에서 처리하고 화면에서는 결과만 표시하게 작성한다. Vue 작성 원칙은 뷰, 데이터, 로직의 분리 [참조]page 101, 102
선언문은 사용 불가능
{ { var a = 10; } }
분기문은 사용 불가능
{ { if (true) {return 100} } }
삼항 연산자는 사용 가능
{{ awesome ? 100 : 0 }}
콧수염 표현식을 사용한 데이터 바인딩 :
HTML이 아닌 일반 텍스트로 해석

{{ message }}

콧수염 표현식 안에서 자바스크립트 연산자 사용 가능

{{ message + "!!!" }}

콧수염 표현식 안에서 자바스크립트 사용 가능하지만
가능한 연산은 인스턴스 속성에서 처리하는 것이 좋다.

{{ message.split('').reverse().join('') }}
복잡한 연산은 인스턴스 computed 속성으로 계산 후 최종 값만 표현
message.split("").reverse().join(" ")
computed 속성을 사용하는 방법:

{{ reversedMessage }}
