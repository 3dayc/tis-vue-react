Vue에서의 Webpack 실습을 위해서는 Vue-CLI 를 사용해야 한다.
      교재 126페이지 참조. 

      # CLI 는 command line interface의 약자.

      step 1. mkdir webpack-test          # 폴더 만들기
           $ ls 

      step 2.              # 폴더 이동
            $ cd webpack-test
            $ ls 
            $ pwd

      step 3. Vue-CLI version 2 다운로드
            $ npm install vue-cli -g
            $ vue -V
            vue/cli 2.9.6

      step 4. Vue 프로젝트 생성
            $ vue init webpack-simple  
            $ ls 

      step 5. Vue 라이브러리 다운로드
            $ ls            # package.json 이 있는지 확인
            $ npm install   # package.json 이 보이는 곳에서 실행

      step 6. Vue 프로젝트 실행
            $ npm run dev    

      step 7. 실습 파일 복사
        Header.vue, Footer.vue, App.vue 파일을 webpack-test/src/ 에 붙여넣기

      step 8. npm run build               # webpack-test/dist/ 에 합쳐진 파일 만들기
      step 9. npm run dev                 # Vue 프로젝트 실행
